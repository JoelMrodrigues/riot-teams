# Summoner Spells Icons

Placez les icônes des summoner spells dans ce dossier avec les noms suivants :

## Icônes nécessaires :

- `flash.png` - Flash (⚡)
- `barrier.png` - Barrier (🛡️)
- `cleanse.png` - Cleanse (✨)
- `heal.png` - Heal (💚)
- `ghost.png` - Ghost (👻)
- `teleport.png` - Teleport (🔮)
- `ignite.png` - Ignite (🔥)
- `exhaust.png` - Exhaust (❄️)
- `smite.png` - Smite (⚔️)

## Format recommandé :
- Taille : 32x32 pixels ou 64x64 pixels
- Format : PNG avec transparence
- Style : Icônes officielles de League of Legends

## Fallback :
Si une icône n'est pas trouvée, l'emoji correspondant sera affiché automatiquement.
